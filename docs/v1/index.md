# SignalStack v1.0 Documentation (Legacy)

Welcome to the documentation for SignalStack v1.0. This version is the foundation release focusing on core event dispatching and webhook delivery.

## Key Features

- **Event Dispatching**: Reliable delivery of events to your subscribed endpoints.
- **Webhooks**: Simple HTTP callbacks for real-time notifications.
- **Basic Triggers**: Define actions based on specific event types.
