# SignalStack v2.0 Documentation (Latest)

Welcome to the documentation for SignalStack v2.0. This major upgrade introduces advanced capabilities for smarter system behavior and improved performance.

## What's New in v2.0

- **Smart Event Routing**: Dynamically route events based on payload content and metadata.
- **Retry Engine**: Configurable retry policies with exponential backoff for failed deliveries.
- **Advanced Analytics**: Real-time insights into event delivery success rates and latency.
- **Event Filtering**: Granular control over which events trigger notifications.

## Getting Started

Head over to the [Getting Started](./getting-started.html) guide to set up your integration with the powerful new features of SignalStack v2.0.
